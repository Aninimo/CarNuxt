# CarNuxt
rental car made with Nuxt, node, tailwindcss, stripe and mongodb

## Setup .env file
```
STRIPE_SECRETE_KEY
```
## Run frontend dev server:
npm run dev

## Run backend server: 
node index.js 

## Technologies used
<ul>
  <li>Nuxt</li>
  <li>TailwindCss</li>
  <li>Prisma</li>
  <li>Stripe</li>
  <li>Node</li>
</ul>
