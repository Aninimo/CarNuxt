# CarNuxt
rental car made with Nuxt, node, tailwindcss, stripe and mongodb

## Demo
https://teste-blond-psi.vercel.app/

## Setup .env file
```
STRIPE
MONGODB
```
## Run frontend dev server:
```
npm run dev
```

## Install deps:
```
npm install
```

## Run backend server:
```
node index.js
```
